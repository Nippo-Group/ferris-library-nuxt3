# フェリス図書館にない資料の利用方法

フェリス図書館にはない図書や雑誌を利用したい場合、主に以下の方法があります。

- 訪問利用（利用したい資料がある図書館を直接訪問する方法）
- 文献取寄せ（図書本体、又は図書・雑誌記事のコピーを、所蔵している図書館から取り寄せる方法）

## フェリス所蔵以外の図書・雑誌を探す方法

利用したい資料をどこの図書館で持っているか調べられる、「全国版OPAC」と呼べる検索サイトがあります。

キーワードなどで検索することで、フェリスを含め全国の図書館が所蔵している資料全体から探すことができます。

＞ [【フェリス所蔵以外の図書・雑誌を探す（全国版OPAC）】の詳細を見る](/search-guide/search-other-libraries)

## 文献取寄せ方法

文献取寄せには、以下の区分があります。いずれも有料サービスになります。

### ・ 現物貸借

図書そのものを取り寄せる方法。雑誌、楽譜、参考図書、音源・映像資料本体の取寄せは原則できません。

### ・ 文献複写

雑誌掲載記事・論文のコピーを取り寄せる方法。

図書や楽譜のコピー取寄せも可能ですが、著作権法による制限があり、全文の取寄せはできません。

＞ [【文献の取寄せ（ILL＝Inter Library Loan）】の詳細を見る](/search-guide/ill)

## 訪問利用方法

公共図書館（国立国会図書館、公立図書館など）は自由に訪問利用できます。

ただし、他大学図書館を訪問利用するには手続が必要です。

＞ [【訪問利用】の詳細を見る](/search-guide/visit-library)
