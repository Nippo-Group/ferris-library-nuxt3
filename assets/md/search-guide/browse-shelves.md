# 直接本棚から探す（ブラウジング）［国際社会学科向け］

図書館の本には1冊ずつラベルが貼ってあり、番号・記号が書かれています。

上段の番号を分類番号と言い、本の主たるテーマに合った番号が付けられています。

本はこの番号順に本棚に並んでいますので、同じ番号の場所には同じテーマの本が集まっています。

必要な資料の番号から本棚を直接探す方法を「ブラウジング」といいます。

分類番号は1冊の本にひとつしか付けられないため、複数のテーマを持つ本はあくまで「主たる」テーマに合った番号が付けられています。

必要な本はブラウジングだけでは見つからないことがありますので、OPAC検索も合わせて使いましょう。

※文庫本・新書はテーマ別に並んでいないので除きます。
