# フェリス所蔵以外の図書・雑誌を探す（全国版OPAC）

## CiNii Books （ アクセスフリー ）

[https://ci.nii.ac.jp/books/](https://ci.nii.ac.jp/books/)

国立情報学研究所が運営する検索サイトです。

全国の大学図書館が所蔵している図書、雑誌の情報を基本に、国立国会図書館等他機関の情報も

取り込んでいます。

## bookplus （契約データベース： 学内利用のみ ）

フェリス女学院大学図書館HP＞電子ブック・データベース（データベース一覧）からアクセス

[https://www.library.ferris.ac.jp/database](https://www.library.ferris.ac.jp/database)

NICHIGAI（日外アソシエーツ）提供の図書内容情報データベースです。

出版情報を基本に国内発行資料について網羅し、タイトルのほか目次や概要も検索範囲に含まれます。

所蔵情報は別に確認する必要がありますが、出版ベースから資料を検索するのに役立ちます。

## NDLサーチ （ アクセスフリー ）

[https://ndlsearch.ndl.go.jp/](https://ndlsearch.ndl.go.jp/)

国立国会図書館のOPACです。国会図書館は国内出版物を広く収集していますので、全国版のOPACと考えることができます。

全国公共図書館の所蔵、CiNii Booksへのリンクも備わっています。

### ※ 神奈川県立図書館　県内横断検索（県内公立図書館版OPAC・ アクセスフリー ）

[https://ufinity.pen-kanagawa.ed.jp/?page_id=722](https://ufinity.pen-kanagawa.ed.jp/?page_id=722)

神奈川県立図書館の検索ツールです。

県内公共図書館の所蔵を一度に検索することができます。

### ※ 東京都立図書館統合検索（都内公共図書館版OPAC・ アクセスフリー ）

[https://uf-pub01.ufinity.jp/cassV3metro/cassrh.do?sc_locale=ja&tenantId=metro&&mode=detail](https://uf-pub01.ufinity.jp/cassV3metro/cassrh.do?sc_locale=ja&tenantId=metro&&mode=detail)

東京都立図書館の検索ツールです。

都内公共図書館等の所蔵を一度に検索することができます。また図書のほか、雑誌記事の検索も可能です。
