# Webから情報を集める（学会・団体サイトリンク集）

Web上の匿名情報（Wikipedia、SNS等）はレポート・論文の参考文献として適切ではありませんが、中には参考文献として使える信頼性の高い情報もあります。

官公庁や自治体、公益性の高い団体のサイト、大学、学会、研究団体など学術機関のサイトなどは信頼性が高いと言えます。

Webの情報を参考文献、引用文献として使う場合は、学科や指導教員の指示する方法（論文作法）にしたがって記載し、信頼性に不安がある時は指導教員に確認してから使用しましょう。

## ■ 国立国会図書館リサーチ・ナビで調べ方を調べる

国立国会図書館リサーチ・ナビは、調べ方を調べる便利なサイトです。

研究テーマの文献探索に参考となる情報やリンクを教えてくれます。

キーワード検索やカテゴリーから閲覧など工夫してうまく使いましょう。
